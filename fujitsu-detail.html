Project - FCCL AI Platform

This was the project of Fujitsu, a Japanese company. I have worked for 3 months in this project. This was an AI based video domain project. In which our track was of VMS integration and metadata support. In this project I did the Gstreamer framework with Python and C / C ++. And along with this, I also worked on different types of video management systems called NxWitness & ZoneMinder on Linux & windows environment.

  • GUI application able to play RTSP(real time feed/live feed) and HTTP feed and display it

    • Description:

    • In this application, I created a desktop application which takes HTTP and RTSP (live feed) stream from NxWitenss.
    • Use Gstreamer to play both HTTP & RTSP stream in two display window
    • Link - Project link

  • Read live stream and draw object on it

    • Description:

    • In this program, I created a C program which reads dummy coordinates from text file.
    • Draw shape according coordinates using Cairo overlay plugin on video frames.
    • Link - Project link

  • Object Detection using deep learning with python on live feed

    • Description:

    • In this program, I created a python program which takes live feed from NxWitness using Gstreamer.
    • Send live stream to a Deep learning based caffemodel for object detection and also display the coordinates
    • Link - Project link

  • Raw coordinate rendering on image

    • Description:

    • In this program, I created a python script which takes sample-coordinate of image from text file coord.txt .
    • Draw all the respective coordinates on the objets of each images.
    • Link - Project link

    • Description:

    • In this program, a C file used to add metadata in h264 stream and send it to UDP port.
